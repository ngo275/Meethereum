# Meethereum
Meetup on Ethereum

## Usage

```bash
git clone https://github.com/ngo275/Meethereum.git
cd Meethereum

# if you use docker-compose
docker-compose build
docker-compose up

# if you don't use docker
cd client
yarn
yarn start
```
